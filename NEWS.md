## ggshow v0.1.7 (Release data: 2022-04-23)


- `ggshow`
  - `imageformat` option added to specify the image format from 'png', 'jp[e]g' and 'svg' (default: 'png')
  - `display` option added to display the image on the cell (default: True)
- `gg` magic
  * `imageformat` option added to specify the image format from 'png', 'jp[e]g' and 'svg' (default: 'png')
